# Staatsbürgerschaftsprüfung Übungstest

A personal project built to better prepare for the Austrian Citizenship Test (*Staatsbürgerschaftsprüfung*).

**Live:** [staatsbuergerschaft-quiz-at.netlify.app](https://staatsbuergerschaft-quiz-at.netlify.app/)

---

## What it does

Interactive quiz covering all 9 Austrian federal states (*Bundesländer*) with realistic exam simulation — official passing conditions, error tracking, and result summaries.

Topics include Austrian history, democracy, federal structure, and state-specific questions.

## Features

- All 9 Bundesländer question sets
- Exam-mode simulation with real pass/fail thresholds
- Wrong-answer memory for targeted practice
- Dark UI, fully responsive
- No backend — single HTML file, works offline

## Usage

Open `index.html` in any browser, or visit the live site above.
